# 📚 Курс по классическому машинному обучению

Добро пожаловать на курс по классическому ML! Этот курс состоит из 14 Jupyter ноутбуков, каждый из которых посвящён отдельной теме.

> **Примечание:** Курс создан исключительно в личных образовательных целях.

## 🎯 Для кого этот курс

- Начинающие в Data Science и ML
- Аналитики данных, желающие углубить знания
- Разработчики, переходящие в ML
- Студенты и специалисты, изучающие ML самостоятельно

## 🚀 Как проходить курс

1. **Последовательно** — ноутбуки построены от простого к сложному
2. **С практикой** — запускайте код, экспериментируйте с параметрами
3. **Конспектируйте** — делайте пометки в ноутбуках
4. **Повторяйте** — перепроходите ноутбуки на своих данных

## 💡 Советы по обучению

- **Не спешите** — лучше глубоко понять одну тему, чем пробежаться по всем

- **Применяйте сразу** — берите свои данные и пробуйте на них
